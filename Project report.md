1. INTRODUCTION :-
   Rootkit Analysis is a cybersecurity project that focuses on detecting and understanding rootkits, which are malicious programs designed to hide their presence in a computer system.  Rootkits allow attackers to gain unauthorized access and control over a system while remaining undetected.
   The goal of this project is to analyze suspicious files and detect whether they show characteristic of known rootkits.

2.OBJECTIVES OF THE PROJECT :-
  The main objectives of this project are :
       *To understand how rootkits works.
       *To build a simple tool to detect suspicious files.
       *To analyze filenames that may represent rootkits.
       *To learn basic malware detection techniques using python.

3.Cyber Security Relevance :-
  Rootkits are one of the most dangerous types of malware because they operate secretly in the background.
  Rootkit analysis helps cybersecurity professionals to :
       *Identify hidden malware
       *protect system from attackers
       *Improve system security
       *Detect suspicious system behavior
  This project demonstrate how basic rootkit detection techniques can be implemented.
