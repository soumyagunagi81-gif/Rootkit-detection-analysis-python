## 1. Introduction
Rootkits are malicious software that hide inside computer systems and allow attackers to gain unauthorized access.  Detecting rootkits is an important part of cybersecurity.

## 2. Objective
The Objective of this project is to develop a simple tool that can detect suspicious rootkit file names.

## 3. Tools And Technologies
- Python
- Flask
- HTML
- CSS

## 4. Implementation
The project uses a Python script to check file names against a list of known suspicious rootkit files.  A web interface built using Flask allows users to enter file names and scan them.

## 5. Working 
1. User enters file name in the web page.
2. Flask sends the file name to the python detection script.
3. The script checks if the file name is in the suspicious file list.
4. The result is displayed on the web page.

## 6. Output
If a suspicious file is detected, the system displays a warning message :
Possible Rootkit Detected.

If the file is safe, the system displays :
File is Safe.
